# 游戏历史排名查看功能

**日期**: 2026-05-27
**状态**: 设计确认

## 概述

游戏详情页新增"历史排名"按钮，点击后在按钮下方弹出面板，展示该游戏在热销榜中的历史排名折线图，支持 7/30/90 天切换。

## 后端

### API 端点

```
GET /api/games/{game_id}/rank-history?days=7
```

- **鉴权**: 无需登录，公开接口
- **查询**: 从 `daily_top_sellers` 表查询该 game_id 在 `date >= today - days` 范围内的记录，按日期升序
- **响应格式**:

```json
{
  "game_id": 1,
  "days": 7,
  "history": [
    {"date": "2026-05-21", "rank": 3},
    {"date": "2026-05-22", "rank": 5}
  ]
}
```

- 无历史记录时返回空 `history: []`，不报 404
- days 参数校验：仅允许 7/30/90

## 前端

### 依赖

- `chart.js` + `vue-chartjs`：折线图渲染

### 组件

**`RankHistory.vue`** — 独立组件，负责：
- 根据传入的 `gameId` 和当前选中的 `days` 请求 API
- 用 Chart.js 渲染折线图（Y 轴反转，第1名在顶部）
- 展示 7天/30天/90天 三个切换按钮
- 加载中 / 无数据状态处理

### 放置位置

- "历史排名"按钮：Hero 模块 `hero-meta` 行，与价格、日期并列
- 弹框面板：按钮下方展开，覆盖在 Hero 模块内部

### 交互

- 点击"历史排名"按钮 → 展开面板，默认加载 7 天数据
- 面板展开后，7/30/90天按钮切换 → 重新请求并更新图表
- 收起方式：
  - 再次点击"历史排名"按钮
  - 点击面板以外的页面任意区域（`onClickOutside`）
- 按钮激活态高亮（面板展开时）

### 图表配置

- **Y 轴**: 反转（`reverse: true`），顶部第1名，底部最低排名
- **X 轴**: 日期，格式 MM-DD
- **颜色**: 霓虹青色（`#00e5ff`）为主色调，填充渐变
- **点**: 显示数据点圆点，悬停展示 tooltip（日期 + 排名）
